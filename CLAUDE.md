# Qantara Development Guide

**Qantara** (قنطرة — Bridge) is Bank Dhofar's Open Banking Platform.

## Quick Reference

### Architecture
- **6 services**: ob-api-server, ob-consent-service, ob-tpp-manager, ob-event-service, ob-portal, ob-sandbox-app
- **Single domain**: `qantara.tnd.bankdhofar.com` (path-based routing via Istio)
- **Mock-first**: All 64 OBIE v4.0 endpoints backed by MockAdapter initially
- **Adapter pattern**: Pluggable backends (mock → corporate → emandate)

### Services

| Service | Language | Port | Path | Purpose |
|---------|----------|------|------|---------|
| ob-api-server | Python/FastAPI | 8000 | `/open-banking/` | OBIE v4.0 API endpoints |
| ob-consent-service | Python/FastAPI | 8000 | (internal) | Consent lifecycle management |
| ob-tpp-manager | Go | 8000 | `/portal-api/` | TPP registration + Keycloak provisioning |
| ob-event-service | Python/FastAPI | 8000 | `/webhooks/`, `/events/` | Event subscriptions + webhook delivery |
| ob-portal | React 18/Mantine | 80 | `/` (catch-all) | Qantara developer portal |
| ob-sandbox-app | Expo/React Native | N/A | N/A | Mock banking app for consent testing |

### Deployment

- **Cluster**: `oci-mct-tnd-rtz`
- **Namespace**: `ob-tnd`
- **Domain**: `qantara.tnd.bankdhofar.com`
- **Keycloak realm**: `open-banking` on existing Keycloak (oci-mct-prod-rtz)
- **Database**: PostgreSQL via CNPG (`qantara-postgres`)
- **Cache**: Redis (`qantara-redis`)
- **Registry**: `harbor.cp.bankdhofar.com/qantara/`

### Branch Strategy
Work on `main` branch.

```bash
cd /vagrant/git/open-banking

git push origin main
```

### Key Patterns
1. **Adapter pattern**: `services/ob-api-server/app/adapters/` — swap mock↔real per endpoint
2. **Consent-gated access**: Every OBIE API call validates consent via ob-consent-service
3. **FAPI headers**: x-fapi-interaction-id on every response (generated by EnvoyFilter or app)
4. **OBIE error format**: All errors return `{ Code, Id, Message, Errors[] }` structure

### OBIE API Prefix
All OBIE endpoints use: `/open-banking/v4.0/{aisp,pisp,cbpii}/...`

```bash

kubectl --context oci-mct-tnd-rtz exec -n ob-tnd qantara-postgres-1 -it -- \
  psql -U qantara -d qantara

kubectl --context oci-mct-tnd-rtz exec -n ob-tnd qantara-redis-0 -- redis-cli
```
